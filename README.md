# United States Cars Analysis
This is an interactive web application developed with Streamlit that allows users to perform an exploratory analysis of a dataset containing car sale listings in the United States.

## Project Description
The application loads a CSV dataset and provides interactive visualizations to analyze the features of vehicles listed for sale. It is designed as a simple tool to explore data visually directly from the browser.

## Funcionalidades
- Interactive data visualization using Plotly.
- Histogram of vehicle mileage (odometer).
- Scatter plot showing the relationship between price and mileage.
- Interactive controls (buttons or checkboxes) to generate charts on demand.
